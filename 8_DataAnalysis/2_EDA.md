# EDA(탐색적 데이터 분석)
> **데이터 타입** : 수치형(연속형, 이산형), 문자형, 범주형(순서형, 명목형), 불리언형  
> **기술 통계** : 평균Mean, 중앙값Median, 최빈값Mod

## 데이터 타입
- 데이터 : seaborn의 iris 데이터, titanic 데이터
- 컬럼별 데이터 타입 확인
    - iris 데이터
        * 수치형: sepal_length	sepal_width	petal_length	petal_width
        * 범주형: species
    - titanic 데이터
        * 수치형(연속형) : age, fare
        * 수치형(이산형) : sibsp, parch
        * 범주형(순서형) : pclass   
        * 범주형(명목형) : sex, class, name, Ticket, cabin, embarked
